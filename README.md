## Ndown V4

## مالجديد ؟
- تحسين في بنيه الكود بحيث يعرض جميع الجودات المتاحة
-  تغيير في التصميم السابق لاصدار الموقع
- إماكنيه إختيار الجودة التي ترديها الان من الموقع
- إزاله الاكواد الغير الهامة بخصوص  تحميل فيديوهات اليوتيوب

## المطلوب 
### في حالة الملف البايثون 
- بايثون مثبت مسبقا
- تنزيل مكبتة yt_dlp

### في حالة موقع الويب 
- استضافه محترمة زي vercel مثلا تمشي الدنيا
- ملف المتطلبات يترفع الاستضافه لانه مهم جدا

## كيفيه التشغيل
- ``` pip3 install yt_dlp ```
- ```python3 app.py ```
